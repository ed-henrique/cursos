## NVM

NVM (Node Version Manager) é uma ferramenta que permite utilizar de forma
intercambiável múltiplas versões diferentes do Node.js. Alguns dos comandos mais
relevantes são:

- `nvm ls-remote`: lista versões disponíveis
- `nvm install VERSION`: instala versão
- `nvm alias default VERSION`: define versão como padrão ao abrir o terminal

Ela suporta um arquivo chamado `.nvmrc`, onde é possível especificar uma versão
do Node.js e facilitar a instalação para colaboradores. Para usar a função do
arquivo, basta rodar `nvm install` no diretório em que ele está.
